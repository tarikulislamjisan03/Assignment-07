# 🛡️ Keen Keeper - Friend Activity Tracker

**Keen Keeper** is a smart activity tracking web application designed to help users manage and monitor their interactions with friends. It provides a clean, organized timeline to log and review various types of communication like Calls, Texts, and Video Chats.

---

## 🚀 Technologies Used

This project is built with a modern tech stack for performance and scalability:

* **React.js**: For building a dynamic and component-based User Interface.
* **Tailwind CSS**: For rapid, utility-first styling and responsive layouts.
* **Lucide React**: For high-quality, consistent iconography.
* **React Router**: To handle seamless navigation between Home, Timeline, and Stats.
* **React Context API**: For efficient global state management of user activities.

---

## ✨ Key Features

### 1. Dynamic Activity Timeline
The app features a real-time timeline that automatically updates whenever a new interaction is logged. Each entry displays the friend's name, the type of activity, and a timestamp, providing a clear history of your social engagements.

### 2. Advanced Multi-Type Filtering
To handle large amounts of data, the application includes a powerful filtering system. Users can use a dropdown menu to instantly toggle between views, filtering activities by **Calls**, **Texts**, or **Videos** to find exactly what they are looking for.

### 3. Fully Responsive & Adaptive UI
Built with a mobile-first approach, the interface is fully responsive. Using Tailwind's flexible grid and flexbox systems, the layout adapts perfectly to any screen size—from large 1400px desktops to small mobile devices—ensuring the content never overflows or breaks.
